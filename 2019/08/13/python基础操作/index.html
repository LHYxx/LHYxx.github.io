python 基础操作宝典

列表List

list.append(x)：添加元素x到末尾
list.extend(list2)：将list2拼接到列表末尾，等于list + list2
list.insert(index, x)：将元素x插入到位置index
list.count(x)：统计x在列表中出现的次数
list.index(x)：x在列表中第一次出现的下标
list.pop(index=-1)：删除列表中index处的值，并返回该值，默认最后一个。
list.remove(x)：删除列表中第一个与x匹配的元素
list.reverse()：反转列表
list.sort(cmp=None, key=None, reverse=False)：对列表进行排序

集合Set

set.add(x):添加元素x，如果已存在，则不进行任何操作
set.update(x):其中x可以是列表、元组、字典等，用来批量将元素添加到集合
set.remove(x):删除元素x，如果没有则报错
set.discard(x):删除元素x，如果没有则不进行任何操作
set.pop():会随机弹出一个元素
set.clear():清空集合
set.copy():返回一个拷贝的集合
set1 & set2:交集，等于set1.intersection(set2)
set1 | set2:并集，等于set1.union(set2)
set1 - set2:差集，等于set1.difference(set2)
set1.issubset(set2):判断set2是否是set1的子集
set1.isdisjoint(set2):判断set1和set2是否有交集

字典Dict

dict.clear()：清空字典
dict.copy()：返回拷贝的字典
dict.get(key, default=None)：返回指定key的value，如果没有该key，则返回default的值
dict.has_key(key)：判断字典中是否有key
dict.items()：以可遍历的形式返回字典中的(key, value)对
dict.update(dict2)：将dict2的内容更新到dict中
dict.pop(key)：删除字典中指定key的值，并返回该值。
dict.popitem()：返回并删除字典中的最后一对(key, value)
dict.fromkeys(seq[, val])：创建一个新字典，以序列 seq 中元素做字典的键，val 为字典所有键对应的初始值

python built-in 函数

abs(a): a的绝对值
divmod(a, b)：把除数和余数运算结果结合起来，返回一个包含商和余数的元组(a // b, a % b)
all(iterable)：函数用于判断给定的可迭代参数 iterable 中的所有元素是否都为 TRUE，如果是返回 True，否则返回 False。元素除了是 0、空、None、False 外都算 True。
any(iterable): 函数用于判断给定的可迭代参数 iterable 是否全部为 False，则返回 False，如果有一个为 True，则返回 True。
ord():字符到ASCII数值或Unicode数值
chr():ASCII数值或Unicode数值到字符
pow(x, y):x的y次方。通过内置的方法直接调用，内置方法会把参数作为整型，而 math 模块则会把参数转换为 float。

filter(function, iterable):函数用于过滤序列，过滤掉不符合条件的元素，返回一个迭代器对象，如果要转换为列表，可以使用 list() 来转换。
该接收两个参数，第一个为函数，第二个为序列，序列的每个元素作为参数传递给函数进行判断，然后返回 True 或 False，最后将返回 True 的元素放到新列表中。

map(function, iterable):会根据提供的函数对指定序列做映射。

zip([iterable, …]):将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的对象。
zip([1, 2, 3], [4, 5, 6]) -> [(1,4), (2,5), (3,6)]
zip(*[(1,4), (2,5), (3,6)]) -> ([1, 2, 3], [4, 5, 6])

round(x [, n]): 四舍五入，保留n位小数。

bin()：返回一个整数 int 或者长整数 long int 的二进制表示，以0b开头（返回字符串型, 0b1010）。
hex()：函数用于将10进制整数转换成16进制，以字符串形式表示，以0x开头。
oct()：函数将一个整数转换成 8 进制字符串。以0o开头。

sorted(iterable, cmp=None, key=None, reverse=False) 函数对所有可迭代的对象进行排序操作。
sort 是应用在 list 上的方法，sorted 可以对所有可迭代的对象进行排序操作。
list 的 sort 方法返回的是对已经存在的列表进行操作，无返回值，而内建函数 sorted 方法返回的是一个新的 list，而不是在原来的基础上进行的操作。
